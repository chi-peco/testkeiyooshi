グループの見分みわけ方かた How to identify the two groups  L6

i-adjective（イ形容詞けいようし） — ends in「い」.

→ 「い」で終わるお。(The last letter is い.)

大おおきい　・　古ふるい　・　重おもい　・　おもしろい　・　楽たのしい

(big / old / heavy / interesting / fun)

na-adjective（ナ形容詞けいようし） — does NOT end in「い」. Adds「な」before a noun.

→ 「い」で終わらおない。名詞めいしの前まえに「な」をつける。(Does not end in い. Add な before a noun.)

静しずか　・　ゆうめい　・　げんき　・　きれい　・　すき

(quiet / famous / energetic / pretty / liked)

⚠️ 例外れいがい Exceptions — ends in「い」but IS a na-adjective

きれい（pretty）・ゆうめい（famous）・きらい（disliked）

→ 「い」で終わるおがナ形容詞けいようし。名詞めいしの前まえは「な」が必要ひつよう。 (They look like i-adjectives but they are na-adjectives. Add な before a noun.)

きれいな花はな ○ correct
きれい 花はな ✖ wrong — missing「な」
肯定形こうていけい Affirmative — "is ~"  Lesson 6

i-adjective（イ形容詞けいようし）

1
Keep「い」. Nothing changes.
→ 「い」のまま。変えかない。(Keep い as-is.)
2
Add です.
→ 「です」をつける。(Add です.)
大きおおきい＋です
ōkii-desu — It is big.

na-adjective（ナ形容詞けいようし）

1
Remove「な」.
→ 「な」を取ると。(Remove な.)
2
Add です.
→ 「です」をつける。(Add です.)
静かしずかな→です
shizuka-desu — It is quiet.
i-adj: N + は + [A-い] + です
na-adj: N + は + [Na-な→] + です
N = noun（名詞めいし）　A = i-adjective（イ形容詞けいようし）　Na = na-adjective（ナ形容詞けいようし）
○ この つくえは 大きいおおきいです。 i-adj: keep「い」+ です.
→ 「い」のまま＋です。(This desk is big.)
× この つくえは 大きおおきです。 Never drop「い」. → 「い」は取っとてはいけません。(Do not remove い.)
○ 私わたしの 町まちは 静しずかです。 na-adj: drop「な」+ です.
→ 「な」を取りと＋です。(My town is quiet.)
× 私わたしの 町まちは 静しずかなです。 Drop「な」before です. → 「な」は取りますと。(Remove な before です.)
否定形ひていけい Negative — "is not ~"  Lesson 6

i-adjective（イ形容詞けいようし）

1
Remove「い」.
→ 「い」を取ると。(Remove い.)
2
Add「くないです」.
→ 「くないです」をつける。(Add くないです.)
大きおおきい→くないです
ōkikunai-desu — not big

na-adjective（ナ形容詞けいようし）

1
Remove「な」.
→ 「な」を取ると。(Remove な.)
2
Add「じゃありません」(casual) or「ではありません」(formal).
→ 「じゃありません」または「ではありません」をつける。
静かしずかな→じゃありません
shizuka-ja arimasen — not quiet
i-adj: N + は + [A-い→く] + ないです
na-adj: N + は + [Na-な→] + じゃありません
N = noun（名詞めいし）　A = i-adjective　Na = na-adjective
○ この かばんは おもくないです。 おもい → おも + くないです.
→ 「い」を取りと「くないです」をつける。(not heavy)
× この かばんは おもいくないです。 Remove「い」first. おもい → おもく (not おもいく).
→ 「い」を先さきに取ると。(Remove い before adding くない.)
名詞修飾めいししゅうしょく Adjective + Noun  Lesson 6

i-adjective（イ形容詞けいようし）

1
Keep「い」. Nothing changes.
→ 「い」のまま。変えかない。
2
Place directly before the noun.
→ 名詞めいしの前まえに置くお。(Put it before the noun.)
古ふるい＋本ほん
furui hon — an old book

na-adjective（ナ形容詞けいようし）

1
Add「な」after the adjective.
→ 形容詞けいようしの後あとろに「な」をつける。(Add な after the adjective.)
2
Place the noun after「な」.
→ 「な」の後あとろに名詞めいしを置くお。(Put the noun after な.)
静かしずかな＋町まち
shizuka-na machi — a quiet town
i-adj: [A-い] + N
na-adj: [Na-な] + N
N = noun（名詞めいし）　A = i-adjective　Na = na-adjective
○ ふじさんは ゆうめいな 山やまです。 na-adj: add「な」before noun.
→ 「な」をつける。(a famous mountain)
× ふじさんは ゆうめい 山やまです。 na-adj needs「な」before noun. → 「な」が必要ひつよう。(な is required.)
過去かこ肯定形こうていけい Past Affirmative — "was ~"  Lesson 11

i-adjective（イ形容詞けいようし）

1
Remove「い」.
→ 「い」を取ると。(Remove い.)
2
Add「かったです」.
→ 「かったです」をつける。(Add かったです.)
大きおおきい→かったです
ōkikatta-desu — It was big.

na-adjective（ナ形容詞けいようし）

1
Remove「な」.
→ 「な」を取ると。(Remove な.)
2
Add「でした」.
→ 「でした」をつける。(Add でした.)
静かしずかな→でした
shizuka-deshita — It was quiet.
i-adj: N + は + [A-い→かった] + です
na-adj: N + は + [Na-な→] + でした
N = noun（名詞めいし）　A = i-adjective　Na = na-adjective
過去かこ否定形ひていけい Past Negative — "was not ~"  Lesson 11

i-adjective（イ形容詞けいようし）

1
Remove「い」.
→ 「い」を取ると。
2
Add「くなかったです」.
→ 「くなかったです」をつける。(Add くなかったです.)
大きおおきい→くなかったです
ōkikunakatta-desu — It was not big.

na-adjective（ナ形容詞けいようし）

1
Remove「な」.
→ 「な」を取ると。
2
Add「じゃありませんでした」.
→ 「じゃありませんでした」をつける。(Add じゃありませんでした.)
静かしずかな→じゃありませんでした
shizuka-ja arimasendeshita — It was not quiet.
i-adj: N + は + [A-い→く] + なかったです
na-adj: N + は + [Na-な→] + じゃありませんでした
N = noun（名詞めいし）　A = i-adjective　Na = na-adjective
て形けい Te-form — connecting adjectives  Lesson 12

i-adjective（イ形容詞けいようし）

1
Remove「い」.
→ 「い」を取ると。
2
Add「くて」.
→ 「くて」をつける。(Add くて.)
大きおおきい→くて
ōkikute — big and ~

na-adjective（ナ形容詞けいようし）

1
Remove「な」.
→ 「な」を取ると。
2
Add「で」.
→ 「で」をつける。(Add で.)
静かしずかな→で
shizuka-de — quiet and ~
i-adj: [A-い→く] + て
na-adj: [Na-な→] + で
A = i-adjective　Na = na-adjective
例外れいがい Exceptions — special cases
⚠️ 「いい」(good) — negative and past use よ- stem

「いい」is used only in affirmative. All other forms use 「よい」 as the base.

→ 「いい」は肯定形こうていけいのみ。否定形ひていけい・過去形かこけいはすべて「よい」を使うつか。(All non-affirmative forms start with よ-.)

Form 形けい いい (good) English
肯定こうてい いいです It is good.
否定ひてい よくないです not good ✖ いくないです
過去かこ（＋） よかったです It was good.
過去かこ（−） よくなかったです not good (past)
て形けい よくて good and ~
